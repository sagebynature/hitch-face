# Changelog

## 0.0.1 - 2026-06-11

Initial release of Hitch Face.

- Added the zero-native desktop widget with animated BMO-style expressions.
- Added local Hitch event ingestion at `127.0.0.1:8888` with session-aware windows.
- Added the Hitch observer adapter and installer for extension/config setup.
- Added source build, test, install, and packaging commands for supported targets.
