Design Spec Decisions

We want to choose the cleanest way to display: Harness Name (omp, codex, hermes, pi, etc.) and Payload Details (tool name, inputs, prompts) without compromising BMO's delightful facial expressions.

  • Minimal Ticker HUD (Option 1): Integrates fully into the virtual screen. Very clean. Keeps BMO compact and self-contained. Shows live context at a glance.
  • Interactive Game Cartridge (Option 2): Fun physical interaction. Fits BMO's gaming console lore perfectly. The cartridge pops in when an action starts, and pops out (slides back down) when it finishes. Hovering displays the detail tooltip.
  • Side Console Log (Option 3): Excellent for debugging. Provides readable formatted text. Slides out to the right using CSS transforms. Best for longer payload outputs.

Tip: In the Side Console and Cartridge views, clicking BMO's buttons or hovering interactive areas triggers animations! Try clicking the red/yellow face buttons.
