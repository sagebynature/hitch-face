How should we display the Hitch metadata?

The Cartridge Pop-in

A Game Boy style cartridge slides into BMO's slot when an event fires. The cartridge label shows the harness name (e.g., "OMP", "Codex") and clicking it reveals a retro pop-up with payload details.

B

Side Console Drawer

Clicking BMO's yellow/red buttons slides open a matching retro developer console drawer from the side, showing a clean, monospaced view of harness, tools, and token cost.

C

Screen HUD & Ticker

Keep the casing clean and utilize the green screen itself. We add a retro ticker tape at the bottom of the screen or a subtle drop-down overlay inside the bezel displaying status and payload details.
